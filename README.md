# Project1
## Доступные команды

### 1. Копирование файлов
Команда позволяет создать копию указанного файла с постфиксом `_copy`.

**Запуск через CLI:**
```powershell
python cli.py copy <имя_файла>

**Запуск через интерактивное меню:**

Запустите python main.py

Выберите номер пункта меню "1. Запустить копировальщик файлов" и введите его в открывшемся окне: 'Введите имя файла, который надо скопировать: '
